# 一、快速排序

> [!info] 快速排序 - OI Wiki  
> OI Wiki 是一个编程竞赛知识整合站点，提供有趣又实用的编程竞赛知识以及其他有帮助的内容，帮助广大编程竞赛爱好者更快更深入地学习编程竞赛  
> [https://oi-wiki.org/basic/quick-sort/](https://oi-wiki.org/basic/quick-sort/)  

### 步骤

1. 确定分界点`x`，可以取`a` 数组在`l`、`r`、`(l+r)/2`的位置，其中`(l+r)/2`可以用`(l+r)>>1`实现

1. 升序情况下，将小于`x`的数放在`x`的左边，将大于`x`的数放在`x`右边

1. 递归运算`x`的左右两个区间

### 区间调整，暴力法

1. 开两个数组存放小于等于`x`的数和大于`x`的数

1. 将两个数组依次放入`a`

- 此方法不太会增加时间复杂度，但增加了空间复杂度

### 区间调整，指针法

1. 让`i`从左往右查找到第一个大于等于`x`的数，让`j`从左往右查找到第一个小于等于`x`的数

1. 交换`a[i]`和`a[j]`

1. 重复直到`i < j`

- 此方法保证`i`左边的数都要小于`x` ，`j` 右边的数都要大于`x`，但是最后一次循环`i`和`j`可能不会相邻，所以要从`j`和`j+1`分割进行递归，以保证右边全大于`x` 而左侧全小于`x`

```C++
void quick_sort(int q[], int l, int r)
{
    if (l >= r) return;

    int i = l - 1, j = r + 1, x = q[(l + r) >> 1];
    while (i < j)
    {
        do i ++ ; while (q[i] < x);
        do j -- ; while (q[j] > x);
        if (i < j) swap(q[i], q[j]);
    }
    quick_sort(q, l, j), quick_sort(q, j + 1, r);
}
```

其中`do while` 语句可以用`while` 语句进行替换：

```C++
while(a[++i]<x);
while(a[--j]>x);
```

当使用`(l, j)`和`(j+1, r)` 作为下一步递归区间时，`x` 可以为`(l+r)>>1` 或`l` ，但是不能为`r` ，同理，当使用`(l, i-1)`和`(i, r)` 作为下一步递归区间时，`x` 可以为`(l+r+1)>>1` 或`r` ，但是不能为`l` ，否则程序可能死循环

### 例题

> [!info] 【模板】排序 - 洛谷  
>  
> [https://www.luogu.com.cn/problem/P1177](https://www.luogu.com.cn/problem/P1177)  

```C++
\#include <iostream>
\#include <cstdio>
using namespace std;
const int N=1e5+10;
void quicksort(int a[],int l,int r){
    if(l>=r)
        return;
    int i=l-1,j=r+1,x=a[(l+r)>>1];
    while(i<j){
        while(a[++i]<x);
        while(a[--j]>x);
        if(i<j) swap(a[i],a[j]);
    }
    quicksort(a,l,j);
    quicksort(a,j+1,r);
}
int main(){
    int n;
    int a[N];
    ios::sync_with_stdio(false);
    cin>>n;
    for(int i=0;i<n;i++)
        cin>>a[i];
    quicksort(a,0,n-1);
    for(int i=0;i<n;i++)
        cout<<a[i]<<' ';
}
```

# 二、归并排序

> [!info] 归并排序 - OI Wiki  
> OI Wiki 是一个编程竞赛知识整合站点，提供有趣又实用的编程竞赛知识以及其他有帮助的内容，帮助广大编程竞赛爱好者更快更深入地学习编程竞赛  
> [https://oi-wiki.org/basic/merge-sort/](https://oi-wiki.org/basic/merge-sort/)  

- 归并排序是稳定的。在原序列中若有两个相同的数，在排序后他们的相对位置不会发生改变，则称这个排序算法是稳定的

### 步骤

1. 定义`mid`赋值为`(l+r)>>1`将数组从中间划分为两个数组并递归

1. 递归至`l==r`返回，此时区间内仅有一个变量

1. 递归返回，合并数组

### 合并数组的步骤

1. 使用两个指针`i`、`j`指向两半数组的开头`l`、`mid+1`

1. 循环比较更小的值放入临时数组`tmp`中，更小的值所在数组的指针向后移动一位

1. 循环在指针`i`或`j`走到末尾时停止，即`i==mid`和`j==r`

1. 将未取出所有值的数组剩下的所有值写入`tmp`数组的尾部，模板中并没有额外判断是哪个数组走到了尾，所以直接使用了两个`while`

1. 将`tmp`数组的所有值写入回原数组，注意原数组需要从`l`开始循环，而`tmp`数组从`0` 开始循环

- 用于合并的两个数组都是递增的，所以指针指向的永远是数组未被放进`tmp`数组中最小的一个

- `k` 用于指向tmp数组的末尾，每次使用`i`、`j`、`k`的时候都要注意++

- 在语句`if (q[i] <= q[j])`中使用`<=` 而不是`<`保证了相同大小的数总是先放入`mid`左侧数组，因此保证了归并排序的稳定性

```C++
void merge_sort(int q[], int l, int r)
{
    if (l >= r) return;

    int mid = (l + r) >> 1;
    merge_sort(q, l, mid);
    merge_sort(q, mid + 1, r);

    int k = 0, i = l, j = mid + 1;
    while (i <= mid && j <= r)
        if (q[i] <= q[j]) tmp[k ++ ] = q[i ++ ];
        else tmp[k ++ ] = q[j ++ ];

    while (i <= mid) tmp[k ++ ] = q[i ++ ];
    while (j <= r) tmp[k ++ ] = q[j ++ ];

    for (i = l, j = 0; i <= r; i ++, j ++ ) q[i] = tmp[j];
}
```

- 使用上述模板时需要定义与`q[]`等大的 `tmp[]`

### 例题

> [!info] 【模板】排序 - 洛谷  
>  
> [https://www.luogu.com.cn/problem/P1177](https://www.luogu.com.cn/problem/P1177)  

```C++

\#include <cstdio>

const int N=1e5+10;
int tmp[N];
void mergesort(int a[],int l,int r){
    if(l>=r) return;
    int mid=(l+r)>>1;
    mergesort(a,l,mid);
    mergesort(a,mid+1,r);
    int k=0,i=l,j=mid+1;
    while(i<=mid && j<=r){
        if(a[i]<=a[j])
            tmp[k++]=a[i++];
        else
            tmp[k++]=a[j++];
    }
    while(i<=mid)
        tmp[k++]=a[i++];
    while(j<=r)
        tmp[k++]=a[j++];
    for(i=l,j=0;i<=r;i++,j++) 
        a[i]=tmp[j];
}
int main(){
    int n;
    int a[N];
    ios::sync_with_stdio(false);
    cin>>n;
    for(int i=0;i<n;i++)
        cin>>a[i];
    mergesort(a,0,n-1);
    for(int i=0;i<n;i++)
        cout<<a[i]<<' ';
}
```

# 三、整数二分

> [!info] 二分 - OI Wiki  
> OI Wiki 是一个编程竞赛知识整合站点，提供有趣又实用的编程竞赛知识以及其他有帮助的内容，帮助广大编程竞赛爱好者更快更深入地学习编程竞赛  
> [https://oi-wiki.org/basic/binary/](https://oi-wiki.org/basic/binary/)  

### 寻找第一个满足条件值

此种情况下 mid 属于左区间，对于区间`[l, r]`定义`mid=(l+r)>>2` ，划分为`[l,mid]`、`[mid+1,r]`两个区间，相当于在迭代中`r=mid` 、`l=mid+1` ，此方法可用于寻找第一个满足条件的值

```C++
int bsearch_1(int l, int r)
{
    while (l < r)
    {
        int mid = (l + r) >> 1;
        if (check(mid)) r = mid;
        else l = mid + 1;
    }
    return l;
}
```

### 寻找最后一个满足条件值

此种情况下 mid 属于右区间，对于区间`[l, r]`定义`mid=(l+r)>>2+1` ，划分为`[l,mid-1]`、`[mid,r]`两个区间，相当于在迭代中`r=mid-1` 、`l=mid` ，此方法可用于寻找最后一个满足条件的值

```C++
int bsearch_2(int l, int r)
{
    while (l < r)
    {
        int mid = (l + r + 1) >> 1;
        if (check(mid)) l = mid;
        else r = mid - 1;
    }
    return l;
}
```

- 在两个模板中，存在`bool check (int x);` 用于判断`mid`是否满足某种条件

- **尤其要注意，在第二种情况**`**mid**`**属于右区间时，计算**`**mid**`**时应加 1 再除 2**

- 第一种情况在计算`mid` 时是向下取整的，所以不会出现进入`[l,mid]`区间时`mid==r` 从而进入死循环，第二种情况反之，在计算`mid` 时向上取整，不会出现进入`[mid,r]`区间时`mid==l` 从而进入死循环

# 四、浮点二分

### 基本步骤

对于区间`[l, r]`定义`mid=(l+r)>>2` ，划分为`[l,mid]`、`[mid,r]`两个区间，相当于在迭代中`r=mid` 、`l=mid`

```C++
double bsearch_3(double l, double r)
{
    const double eps = 1e-6;
    while (r - l > eps)
    {
        double mid = (l + r) / 2;
        if (check(mid)) r = mid;
        else l = mid;
    }
    return l;
}
```

- 注意 double 类型无法使用位运算除 2

- 在模板中，存在`bool check (int x);` 用于判断`mid`是否满足某种条件

- `esp`代表精度，根据题目要求而定，当`l`和`r`的差值小于等于`esp`时，此时可以认为`l`和`r`是同一个数，迭代结束。一般来说，要求精确到 n 位小数，可设置`esp`为`1e(-n-2)`

### 一个简单的求 n 的平方根的程序

保证 $0<=n<=10000$ ，精确到两位小数

```C++
\#include <iostream>
\#include <iomanip>
using namespace std;
int main() {
    double n,l=0,r=100,esp=1e-4;
    cin>>n;
    while(r-l>esp){
        double mid=(l+r)/2;
        if(mid*mid>n) r=mid;
        else l=mid;
    }
    cout<<fixed<<setprecision(2)<<l<<endl;
}
```

# 五、高精度加法

常考的高精度题：1、A + B 2、A - B 3、A * a 4、A / a  
其中大写的 A 和 B 位数上限大约为 $10^6$，小写的 a 一般不超过整型范围，以下内容讲的是第一种情况，高精度加高精度。

### 基本思想

用数组倒序分别存放大整数的每一位，例如 `a[100]={9,8,7,6,5,4,3,2,1}`存放的就是 $123,456,789$这么一个数，倒序存放方便进位时添加数字

模拟列竖式计算是高精加的基本思想，每一位数字由另外两个数对应位数的相加，再加上上一位数的进位数

```C++
vector<int> add(vector<int> &A, vector<int> &B)
{
    if (A.size() < B.size()) return add(B, A);

    vector<int> C;
    int t = 0;
    for (int i = 0; i < A.size(); i ++ )
    {
        t += A[i];
        if (i < B.size()) t += B[i];
        C.push_back(t % 10);
        t /= 10;
    }

    if (t) C.push_back(t);
    return C;
}
```

# 六、高精度减法

### 基本思想

和高精度加法类似，进位所用的`t`变成了借位，最后将多余的0弹出，如果答案vector为空就再加上0，如果A有可能小于B，那就加上一个 `bool ifNeg` 函数判断正负

```C++
vector<int> sub(vector<int> &A, vector<int> &B)
{
    vector<int> C;
    for (int i = 0, t = 0; i < A.size(); i ++ )
    {
        t = A[i] - t;
        if (i < B.size()) t -= B[i];
        C.push_back((t + 10) % 10);
        if (t < 0) t = -1;
        else t = 0;
    }

    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

bool ifNeg(VI &a,VI &b){
    if(a.size()>b.size())
        return 0;
    else if(a.size()<b.size())
        return 1;
    for(int i=a.size();i>=0;i--){
        if(a[i]>b[i]) return 0;
        if(a[i]<b[i]) return 1;
    }
    return 0;
}
VI sub(VI &a,VI &b){
    if(ifNeg(a,b)==1)
        return sub(b,a);

    int t=0;
    for(int i=0;i<a.size();i++){
        t+=a[i];
        if(i<b.size()) t-=b[i];
        c.push_back((t+10)%10);
        if(t<0) t=-1;
        else t=0;
    }
    while(c.size()>0&&c.back()==0) c.pop_back();
    if(c.empty()) c.push_back(0);
    return c;
}

    for(int i=a.size()-1;i>=0;i--)
        printf("%d",a[i]);
    printf("\n");
}

    for(int i=input.size()-1;i>=0;i--)
        a.push_back(input[i]-'0');
    return a;
}

    VI a,b;
    scan(a);
    scan(b);
    VI c=sub(a,b);

# 七、高精度乘法

### 高精度乘低精度

一般来说在高精度每一位数乘完被乘数时，long long 类的临时变量`t`仍然能存放，所以直接一次性让 A 的每一位数和 b 相乘得到`t`，`t%10`即为当前位数，`t/10` 即为下一位数的进位。要注意由于进位`t`有可能会非常大，所以要等`t`循环至 0 后再结束循环，最好要去除多余的 0

```C++
vector<int> mul(vector<int> &A, int b)
{
    vector<int> C;

    int t = 0;
    for (int i = 0; i < A.size() || t; i ++ )
    {
        if (i < A.size()) t += A[i] * b;
        C.push_back(t % 10);
        t /= 10;
    }

    while (C.size() > 1 && C.back() == 0) C.pop_back();

    return C;
}
```

- 注意，高精度的所有算法中，刚进入循环时，`t` 都不会被直接赋值，而是使用`+=`或`-=` 运算，否则就销毁了上一位保留的进位

```C++
\#include <iostream>
\#include <algorithm>
\#include <string>
using namespace std;
const int MAXLEN=4000;
string multi(string a,string b){
    int res[MAXLEN];//用于储存未进位的积
    for(int i=0;i<MAXLEN;i++)
        res[i]=0;
    int lenA=a.size();
    int lenB=b.size();
    for(int i=0;i<lenA;i++)
        for(int j=0;j<lenB;j++)
            res[i+j]+=int(a[lenA-i-1]-'0')*int(b[lenB-j-1]-'0');
    int it=0;
    int minloop=lenA+lenB-2;
    while(res[it]>=10 || it<=minloop){
        res[it+1]+=res[it]/10;
        res[it]%=10;
        it++;
    }
    while(res[it]==0)
        it--;
    string ans="";
    for(int i=it;i>=0;i--)
        ans+=res[i]+'0';
    if(ans=="")
        return "0";
    return ans;
}
int main(){
    string a,b;
    cin >> a >> b;
    cout<< multi(a,b) <<endl;
    return 0;
}
```

基本思路：

列竖式计算，得到没有任何进位的结果，也就是说每一位数字都有可能是一个几十上百的数，所以用res数组储存，最后再进位，注意0做乘数的判断

例如：

  

$$\begin{array}{r}  
& & && 1 & 5 & 9 & 7 & 5 & 3 \\  
\times & && && & 1 & 2 & 3 & 4 \\  
\hline  
& && & 6 & 3 & 9 & 0 & 1 & 2 \\  
& && 4 & 7 & 9 & 2 & 5 & 9 \\  
& &3 & 1 & 9 & 5 & 0 & 6 \\  
+ & 1 & 5 & 9 & 7 & 5 & 3 \\  
\hline  
& 1 & 8 & 14 & 29 & 22 & 14 & 11 & 10 & 2 \\  
\end{array}$$

最后从个位开始进位并模10即可

  

# 八、高精度除低精度

### 基本思想

高精度除法的算法和前面三种高精度略有不同，基本思路是模拟列竖式除法计算，先从高位开始除，循环中，先获取最高位，加在乘10倍的上一位的余数，除b得到当前位，在模b得到当前位的余数，循环结束，`r`即为$A\div b$，在函数内引用`r`以返回余数。

由于从高位开始计算，最后需要反转答案数组，然后去除高位0

```C++
vector<int> div(vector<int> &A, int b, int &r)
{
    vector<int> C;
    r = 0;
    for (int i = A.size() - 1; i >= 0; i -- )
    {
        r = r * 10 + A[i];
        C.push_back(r / b);
        r %= b;
    }
    reverse(C.begin(), C.end());
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

# 九、前缀和&差分

### 前缀和

若有数组 $a_0,a_1,a_2,a_3......a_i$，则 $a_i$ 数组的前缀和数组为 $s_i=\sum_{n=1}^{i} a_n$，对数组区间求和时使用 $\sum_{n=l}^{r} a_n=s_r-s_{l-1}$ 快速计算是前缀和的主要用法

需要注意的是，数组需要从`a[1]`开始存，前缀和数组首项`s[0]=0`，保证在求和时`l-1`不会越界

二维前缀和类似，用于快速求从某点到某点矩形区域内的所有元素的和

### 差分

差分是前缀和的逆运算，若有数组$a_0,a_1,a_2,a_3......a_i$，构造 $b_i$ 使得 $a_i=\sum_{n=1}^{i} b_n$ 。用$a_i$的差分$b_i$存放数据时，若要对数组批量加减，时间复杂度会由$O(n)$降低至$O(1)$，但访问$a_i$的时间复杂度会从$O(1)$提高至$O(n)$，这是差分的主要用法

当需要给数组`a[]`$[l,r]$内的元素加c时，差分数组的操作：`b[l]+=c;b[r+1]-=c;`

二维差分类似，构造差分矩阵后可以快速对矩形区域内的元素批量加减

取用时：`pre[k][l]-pre[i-1][l]-pre[k][j-1]+pre[i-1][j-1]`

# 十二、整数离散化

如果有数量不多但是数据范围很大的数字，将其离散化便是按序将其映射到一个下标，离散化后需要使用分求出对应数字的下标，以下模板包含了离散化和查找下标两种操作

![[image.png]]

```C++
// 离散化数组
vector<int> alls; // 存储所有待离散化的值
sort(alls.begin(), alls.end()); // 将所有值排序
alls.erase(unique(alls.begin(), alls.end()), alls.end());   // 去掉重复元素

// 二分求出x对应的离散化的值
int find(int x) // 找到第一个大于等于x的位置
{
    int l = 0, r = alls.size() - 1;
    while (l < r)
    {
        int mid = l + r >> 1;
        if (alls[mid] >= x) r = mid;
        else l = mid + 1;
    }
    return r + 1; // 映射到1, 2, ...n
}
```

注意去重的实现`alls.erase(unique(alls.begin(), alls.end()), alls.end());`

函数`unique()`可以把数组中所有重复元素放在数组的末尾，然后返回这些重复元素的开头，最后用`erase()`，抹去所有重复元素

如 1 2 2 3 3 3 4 4 4 4 在经过函数`unique()`操作后变为 1 2 3 4 2 3 3 4 4 4

# 十三、区间合并

将有交集的区间进行合并的算法，基本步骤如下：

1. 按左端点排序区间

1. 每次维护一段区间，将结尾标记放在初始区间的结尾，若下一个区间的开始小于标记，则将结尾更更新为该区间的结尾

  

![[image 1.png]]

```C++
// 将所有存在交集的区间合并
void merge(vector<PII> &segs)
{
    vector<PII> res;

    sort(segs.begin(), segs.end());

    int st = -2e9, ed = -2e9;
    for (auto seg : segs)
        if (ed < seg.first)
        {
            if (st != -2e9) res.push_back({st, ed});
            st = seg.first, ed = seg.second;
        }
        else ed = max(ed, seg.second);

    if (st != -2e9) res.push_back({st, ed});

    segs = res;
}
```

### 例题

> [!info] 火烧赤壁 - 洛谷  
>

```C++
\#include <iostream>
\#include <algorithm>
\#include <vector>
using namespace std;
const int N=2e4+5;
struct sec{int h,t;} a[N];
bool cmp(sec a,sec b){return a.h<b.h;}
int main() {
  int n;
  cin>>n;
  for(int i=0;i<n;i++)
    cin>>a[i].h>>a[i].t;
  sort(a,a+n,cmp);
  int hh=a[0].h,tt=a[0].t;
  vector<sec> b;
  for(int i=1;i<n;i++){
    if(a[i].h<=tt)
      tt=max(tt,a[i].t);
    else{
      sec tmp={hh,tt};
      b.push_back(tmp);
      hh=a[i].h;
      tt=a[i].t;
    }
  }
  sec tmp={hh,tt};
  b.push_back(tmp);
  int ans=0;
  for(int i=0;i<b.size();i++)
    ans+=b[i].t-b[i].h;
  cout<<ans<<endl;
}
```

区间 a 合并的结果为 b，最后统计 b 中每一段的差值
