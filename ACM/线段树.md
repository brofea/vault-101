# 什么是线段树？

> [!info] 线段树 - OI Wiki

> [https://oi-wiki.org/ds/seg/](https://oi-wiki.org/ds/seg/)  

> 线段树是算法竞赛中常用的用来维护 **区间信息** 的数据结构。
> 
> 线段树可以在 $O(\log N)$ 的时间复杂度内实现单点修改、区间修改、区间查询（区间求和，求区间最大值，求区间最小值）等操作。

## 优势

|   |   |   |   |   |
|---|---|---|---|---|
||区间求和|区间最大|区间修改|单点修改|
|前缀和数组|〇|$\times$|$\times$|$\times$|
|树状数组|〇|需扩展|需扩展|$\times$|
|线段树|〇|〇|〇|〇|

## 构造

如下图是一个区间长度为 10 的线段树，线段树除了最后一层以外是一棵满二叉树，但并不是一棵完全二叉树，因为最后一层不是按照从左到右排列的

![[QQ_1736226184013.png]]

  

二叉树这样计算子节点的区间，若有父节点表示区间为 $[\text L,\text R]$ 那么 $\text{mid}=\lfloor(\text L+\text R)/2\rfloor$，子节点的区间为 $[\text L,\text {mid}]$ 和 $[\text {mid}+1,\text R]$

虽然它不是完全二叉树，不过仍然可以用一个一位数组来存储，原理和一维数组存储完全二叉树的方式一致，对于节点 $x$ ——

- 父节点是 $\lfloor x/2 \rfloor$，即 `x>>1`

- 左儿子是 $2x$，即 `x<<1`

- 右儿子是 $2x+1$，即 `x>>1|1`

# 线段树的操作

基础线段树有五个操作：

- `pushup()` 和 `pushdown()`，根据子节点维护父节点和根据父节点维护子节点

- `build()` 初始化/建立一颗线段树

- `modify()` 修改线段树，分为单点和区间修改

- `query()` 区间查询

## Pushup

该操作就是用当前节点的子节点更新当前节点

Pushup 本身不需要任何递归进行，因为使用 Pushup 操作的时候都是在递归中使用，每次结束这一层的计算之前（也就是下面的层都已经计算完毕了）就更新当前层的值，所以只要找到当前节点的子节点然后更新就可以了

## 建树

从根节点开始建树，**一共四步**，伪代码如下：

- 函数 `build(u,l,r)` 传入三个参数，节点编号 `u` 区间左右端点 `l,r`
    
    1. 记录区间信息`tr[u].l = l`，`tr[u].r = r`
    
    1. 区间若仅有一个数，返回，无须继续递归
    
    1. 递归子节点，`build(u<<1, l, mid); build(u<<1|1, mid+1, r);`
    
    1. 更新父节点，`pushup(u)`
    

## 区间查询

若需要查询 [L,R]，对于区间 [A,B] 有**三种情况**

1. 如果 [A,B] $\in$ [L,R]，那么直接返回

1. 如果 [A,B] 与 [L,R] 有交集，那么递归

1. 如果 [A,B] 与 [L,R] 无有交集，那么忽略这种情况

比方说，要在区间 [5,9] 内查询最大值——

![[QQ_1736229746588.png]]

也就是说，只要返回 5、[6,8]、9 的最大值就能找到 [5,9] 区间的最大值

- **为什么查询操作一定是** $O(\log n)$ **级别的？**
    
    在上述查询递归时的三种情况中，第一第三种都直接返回了，第二种情况中分为图中的三种
    
    ![[QQ_1736231055989.png]]
    
    其中第一种情况下，mid 如果在 L 左侧，左侧直接返回，右侧仍是第一种情况，mid 如果在 L 右侧，右侧直接返回，左侧仍是第一种情况，所以时间复杂度是 $O(\log n)$
    
    第二种情况和第一种类似，时间复杂度是$O(\log n)$
    
    第三种情况下，mid 如果落在 [L,R] 外面，一侧直接返回，一侧仍是第三种情况，直到mid 落在 [L,R] 内之后，此后再分不可能回到第三种情况，左右两侧会变成第一第二种情况，让 mid 落在区间内最坏情况下就是 $O(\log n)$
    
    所以查询操作是 $O(\log n)$ 级别的
    

## 单点修改

单点修改的伪代码：

- 函数 `modify(u,x,v)` 传入节点编号 `u` ，需要将点 `x` 的值修改为 `v`
    
    1. 如果查找到叶节点，即 `u` 的区间是 `x` 一个点那么赋值并返回
    
    1. 否则递归左右两个区间，计算 mid
        
        1. 如果 `x<=mid` 那么查询左儿子
        
        1. 否则查询右儿子
        
    
    1. Pushup，将下方计算好的信息更新自己
    

要注意的是，递归左右两个区间判断条件不能错，因为左右儿子的区间是`[l,mid]`和 `[mid+1,r]`

## 例题（区间查单点改）

### 区间最大值

题目：[https://www.luogu.com.cn/problem/P1198](https://www.luogu.com.cn/problem/P1198)

特点：一个动态长度的序列，建树时建造空树

题解：[洛谷 P1198 最大数](https://www.notion.so/P1198-17483793b8f680b2a871fe5dfd7f0db1?pvs=21)

- 代码如下（注意建空树的操作没有Pushup）
    
    ```C++
    \#include <bits/stdc++.h>
    using namespace std;
    \#define ll long long
    \#define mp make_pair
    const int N = 2e5+10;
    
    int m,D;
    struct Node{
      int l,r,val;
    }tr[N*4];
    
    void pushup(int u){
      tr[u].val=max(tr[u<<1].val,tr[u<<1|1].val);
    }
    
    // 建树
    void build(int u,int l,int r){
      // 1、记录区间信息
      tr[u].l=l;
      tr[u].r=r;
      tr[u].val=0;
      // 2、如果区间仅有一个数返回
      if(l==r) return;
      // 3、递归建树
      int mid = (l+r)>>1;
      build(u<<1,l,mid);
      build(u<<1|1,mid+1,r); 
      // 4、由于是空树，不需要pushup操作
    }
    
    // 查询[l,r]区间的最大值
    int query(int u,int l,int r){
      // 情况1:被包含
      if(l<=tr[u].l && tr[u].r<=r)
        return tr[u].val;
      // 情况2:有交集
      int mid = (tr[u].l+tr[u].r)>>1;iint
      int tmp=INT_MIN;
      if(l<=mid) tmp=max(tmp,query(u<<1,l,r));
      if(mid<r) tmp=max(tmp,query(u<<1|1,l,r));
      return tmp;
    }t
    
    // 将单点x修改为val
    void modify(int u,int x,int val){
      if(tr[u].l==x && tr[u].r==x){
        tr[u].val=val;
        return;
      }
      int mid = (tr[u].l+tr[u].r)>>1;
      if(x<=mid) modify(u<<1,x,val);
      else modify(u<<1|1,x,val);
      pushup(u);
    }
    
    int main() {
      ios::sync_with_stdio(0);
      cin.tie(NULL);
      cout.tie(NULL);
    
      int n=0;    // 当前序列长度
      int last=0; // 保存上一次查询的值
      cin>>m>>D;
      build(1,1,m);
    
      char op;
      int x;
      while(m--){
        cin>>op>>x;
        if(op=='Q'){
          // 保证x合法，不大于n且不为0
          x=x>n?n:x;
          if(x==0) last=0;
          else last=query(1,n-x+1,n);
          cout<<last<<endl;
        }else{
          // 保证tmp合法，不溢出且不为负
          ll tmp=((ll)last+(ll)x)%D;
          tmp=tmp<0?tmp+D:tmp;
          modify(1,n+1,tmp);
          n++; 
        }
      }
      return 0;
    }
    ```
    

### **区间最大子段和**

题目：[https://www.spoj.com/problems/GSS3/](https://www.spoj.com/problems/GSS3/)  
[https://www.luogu.com.cn/problem/SP1716](https://www.luogu.com.cn/problem/SP1716)

特点：子段和需要的树要维护的信息很多，较复杂，好再并没有涉及到区间修改

题解：[SPOJ GSS3 你能回答这些问题吗III](https://www.notion.so/SPOJ-GSS3-III-17483793b8f680639602e1f58320416a?pvs=21)

- 代码如下（注意树的信息和查询方式）
    
    ```C++
    \#include <bits/stdc++.h>
    using namespace std;
    \#define ll long long
    \#define mp make_pair
    const int N = 50005;
    
    int n,a[N];
    struct Node{
      int l,r;  // 区间端点
      int tmax; // 字段和
      int lmax; // 前缀和
      int rmax; // 后缀和
      int sum;  // 区间和
    }tr[N*4];
    
    inline void pushup(Node &u,Node &l,Node &r){ // 可以传入l和r更新u
      u.sum=l.sum+r.sum;
      u.tmax=max(l.rmax+r.lmax,max(l.tmax,r.tmax));
      u.lmax=max(l.lmax,l.sum+r.lmax); 
      u.rmax=max(r.rmax,r.sum+l.rmax);
    }
    inline void pushup(int u){  // 一个参数，用于在build和modify中操作节点u
      pushup(tr[u],tr[u<<1],tr[u<<1|1]);
    }
    void build(int u,int l,int r){
      tr[u].l=l;
      tr[u].r=r;
      if(l==r){
        tr[u].lmax=tr[u].rmax=tr[u].tmax=tr[u].sum=a[l];
        return;
      }
      int mid=(l+r)>>1;
      build(u<<1,l,mid);
      build(u<<1|1,mid+1,r);
      pushup(u);
    }
    // 重点在区间查询
    Node query(int u,int l,int r){
      if(l<=tr[u].l && tr[u].r<=r)
        return tr[u];
      int mid=(tr[u].l+tr[u].r)>>1;
      // 第一种情况，查询的区间完全在左子区间
      if(r<=mid) return query(u<<1,l,r);
      // 第二种情况，查询的区间完全在右子区间
      else if(l>mid) return query(u<<1|1,l,r);
      // 第三中情况，查询两侧然后pushup
      else {
        Node left=query(u<<1,l,r);
        Node right=query(u<<1|1,l,r);
        Node res;
        pushup(res,left,right);
        return res;
      }
    }
    void modify(int u,int x,int v){
      if(tr[u].l==x && tr[u].r==x){
        tr[u].lmax=tr[u].rmax=tr[u].tmax=tr[u].sum=v;
        return;
      }
      int mid=(tr[u].l+tr[u].r)>>1;
      if(x<=mid) modify(u<<1,x,v);
      else modify(u<<1|1,x,v);
      pushup(u);
    }
    
    int main() {
      ios::sync_with_stdio(0);
      cin.tie(NULL);
      cout.tie(NULL);
     
      cin>>n;
      for(int i=1;i<=n;i++)
        cin>>a[i];
      build(1,1,n);
    
      int q,op,x,y;
      cin>>q;
      while(q--){
        cin>>op>>x>>y;
        if(op==1){
          if(x>y) swap(x,y); // 注意x>y是可能的
          cout<<query(1,x,y).tmax<<endl;
        } else {
          modify(1, x, y);
        }
      }
      return 0;
    }
    ```
    

### 区间最大公约数

题目：[https://www.luogu.com.cn/problem/P10463](https://www.luogu.com.cn/problem/P10463)

特点：最大公约数维护稍麻烦，虽然涉及到区间修改，但是可以用差分数组巧妙解决

题解：[洛谷 P10463 区间最大公约数](https://www.notion.so/P10463-17583793b8f6808394e4e3483b432f6a?pvs=21)

- 代码如下
    
    ```C++
    \#include <bits/stdc++.h>
    using namespace std;
    \#define ll long long
    \#define mp make_pair
    const int N = 500010;
    ll n,m,a[N];
    struct Node{
      int l,r;
      ll gcd;  // 区间最大公约数
      ll sum;  // 区间和，所有数组都要ll，区间信息可以用int
    }tr[4*N]; // 记得到四倍空间
    ll gcd(ll a,ll b){
      return b?gcd(b,a%b):llabs(a); // 要考虑负数，数学中最大公约数没有负数
    }
    void pushup(Node &u,Node &l,Node &r){
      u.gcd=gcd(l.gcd,r.gcd);
      u.sum=l.sum+r.sum;
    }
    void pushup(int u){
      pushup(tr[u],tr[u<<1],tr[u<<1|1]);  // 所有的访问子节点都是<<
    }
    void build(int u,int l,int r){
      // 1、记录区间端点
      tr[u].l=l;    //记得先记录区间端点信息
      tr[u].r=r;
      // 2、如果找到叶节点，输入信息并返回
      if(l==r){
        tr[u].gcd=a[l]-a[l-1];
        tr[u].sum=a[l]-a[l-1];
        return;
      }
      // 3、递归求子区间
      int mid=(l+r)>>1;   // 所有的求mid都是>>
      build(u<<1,l,mid);
      build(u<<1|1,mid+1,r);
      // 4、子区间全部计算完成，pushup当前节点，给当前节点赋值
      pushup(u);
    }
    void modify(int u,int x,ll v){ // v是数值不是区间要ll
      // 1、如果找到叶节点就修改值并返回
      if(tr[u].l==x && tr[u].r==x){
        tr[u].sum+=v;
        tr[u].gcd=abs(tr[u].sum); // 处理负数
        return;    // 找到叶节点记得立刻返回
      }
      // 2、如果没有找到递归找子区间
      int mid=(tr[u].l+tr[u].r)>>1;
      if(x<=mid) modify(u<<1,x,v);
      else modify(u<<1|1,x,v); 
      // 3、下方完成计算，pushup当前节点
      pushup(u); 
    }
    Node query(int u,int l,int r){
    	// 情况1、区间完全被包含
      if(l<=tr[u].l && tr[u].r<=r){
        return tr[u];   // 在区间内了就返回
      }
      // 情况2和3、区间在左右子区间内
      int mid = (tr[u].l+tr[u].r)>>1;
      if(r<=mid) return query(u<<1,l,r);
      else if (l>mid) return query(u<<1|1,l,r);
      // 情况4、区间横跨左右子区间
      else{
        Node left=query(u<<1,l,r);
        Node right=query(u<<1|1,l,r);
        Node res;
        pushup(res,left,right);
        return res;  
      }
    }
    signed main() {
      ios::sync_with_stdio(0);
      cin.tie(NULL);
      cout.tie(NULL);
      cin>>n>>m;
      a[0]=0;  // 最好写
      for(int i=1;i<=n;i++)
        cin>>a[i];
      build(1,1,n); // 记得建树
      char op;int l,r;ll d; // d是数值不是区间要ll
      while(m--){
        cin>>op;
        if(op=='Q'){
          cin>>l>>r;
          if(l==r){
            ll res=abs(query(1,1,l).sum); // 处理负数
            cout<<res<<endl;
            continue;
          }
          Node left=query(1,1,l);
          Node right=query(1,l+1,r);
          ll res=gcd(left.sum,right.gcd);
          cout<<res<<endl;
        } else {
          cin>>l>>r>>d;
          modify(1,l,d);
          if(r+1<=n) modify(1,r+1,-d);
        }
      }
      return 0;
    }
    ```
    

## Pushdown & 懒惰标记

该操作就是用当前节点更新子节点

更新的内容就是懒惰标记（Lazy Tag），为了修改区间，用单点修改的最坏情况下需要修改整一棵树，时间复杂度是 $O(4n)$，这是我们不能接受的

懒标记的思想来源于区间查询，如果区间完全被包含在查询的区间内就直接结束递归，那么区间操作也可以如此，如果递归到该区间完全被包含在需要修改的区间，就在当前节点打上懒标记，等到查询的时候再将这个懒惰标记向下传递

![[QQ_1736319792102.png]]

例如，如果要维护区间和 `sum`，那么可以给数组使用的结构体添加一个 `add` 信息，在 pushdown 时的操作就是这么五步：（传入根节点 `Node &root`，左右子节点 `Node &left`和 `Node &right`）

- `left.add += root.add`

- `right.add += root.add`

- `left.sum += (left.R-left.L+1)*root.add`

- `right.sum += (right.R-right.L+1)*root.add`

- `root.add=0`

![[QQ_1736320523799.png]]

  

## 区间修改

和单点修改差不多，但是注意，区间修改时如果修改的区间横跨两个子节点，那么两个都要访问，而不是像单点修改一样，比单点修改多了一步，在递归之前需要 Pushup 把标记传递下去

区间修改的伪代码：

- 函数 `modify(u,l,r,v)` 传入节点编号 `u` ，需要将区间 `[l,r]` 的值修改为 `v`
    
    1. 如果查找到区间完全被包含于 `[l,r]` 返回信息
    
    1. Pushdown 将自己的懒标记传递到下方供计算使用
    
    1. 否则递归左右两个区间，计算 `mid` ，递归计算两个子区间，然后合并结果
    
    1. Pushup 用下方计算好的信息更新自己
    

## 例题（区间查区间改）

### 区间总和（区间加）

题目：[https://www.luogu.com.cn/problem/P3372](https://www.luogu.com.cn/problem/P3372)

特点：区间查区间改的模板题

给定 n 个数 m 个操作，输入 1 x y k 将区间 [x,y] 所有的数加上 k，输入 2 x y 查询 [x,y] 区间和

- 代码如下
    
    ```C++
    \#include <bits/stdc++.h>
    using namespace std;
    \#define ll long long
    \#define mp make_pair
    const int N = 1e5+10;
    int n,a[N];
    struct Node{
      int l,r;
      ll sum;
      ll add;
    }tr[N*4];
    void pushup(int u){
      tr[u].sum=tr[u<<1].sum+tr[u<<1|1].sum;
    }
    void pushdown(int p){
      Node &u=tr[p],&l=tr[p<<1],&r=tr[p<<1|1];
      if(u.add!=0){
        l.sum+=(l.r-l.l+1)*u.add;
        r.sum+=(r.r-r.l+1)*u.add;
        l.add+=u.add;
        r.add+=u.add;
        u.add=0;
      } 
    }
    void build(int u,int l,int r){
      tr[u].l=l;
      tr[u].r=r;
      if(l==r){
        tr[u].sum=a[l];
        return;
      }
      int mid=(l+r)>>1;
      build(u<<1,l,mid);
      build(u<<1|1,mid+1,r);
      pushup(u);
    }
    void modify(int p,int l,int r,ll d){
      Node &u=tr[p];
      if(l<=u.l && u.r<=r){
        u.sum+=(u.r-u.l+1)*d;
        u.add+=d;
        return;
      }
      pushdown(p);
      int mid=(u.l+u.r)>>1;
      if(l<=mid) modify(p<<1,l,r,d);
      if(r>mid) modify(p<<1|1,l,r,d);
      pushup(p);
    }
    ll query(int p,int l,int r){
      Node &u=tr[p];
      if(l<=u.l && u.r<=r)
        return u.sum;
      pushdown(p);
      int mid=(u.l+u.r)>>1;
      ll res=0;
      if(l<=mid) res+= query(p<<1,l,r);
      if(r>mid) res+= query(p<<1|1,l,r);
      return res;
    }
    int main() {
      ios::sync_with_stdio(0);
      cin.tie(NULL);
      cout.tie(NULL);
      int m;
      cin>>n>>m;
      for(int i=1;i<=n;i++)
        cin>>a[i];
    ```

### 区间总和（区间加区间乘）

题目：[https://www.luogu.com.cn/problem/P3373](https://www.luogu.com.cn/problem/P3373)

特点：区间查区间改的模板题

题解：[洛谷 P3373 线段树2](https://www.notion.so/P3373-2-17583793b8f6806d82fcd2929f03307c?pvs=21)

- 代码如下
    
    ```C++
    \#include <bits/stdc++.h>
    using namespace std;
    \#define ll long long
    \#define mp make_pair
    const int N = 1e5+10;
    
    int n,q,m,a[N];
    struct Node{
      int l,r;
      int sum;
      int add,mul;
    }tr[N*4];
    void pushup(int u){
      // 任何有设计加法乘法都先强制转换为ll
      tr[u].sum=((ll)tr[u<<1].sum+tr[u<<1|1].sum)%m;
    }
    // 关键在于两个互相影响的懒标记如何维护：calc和pushdown函数
    void calc(Node &t,int add,int mul){
      t.sum=((ll)t.sum*mul+(ll)(t.r-t.l+1)*add)%m;
      t.mul=(ll)t.mul*mul%m;
      t.add=((ll)t.add*mul+add)%m;
    }
    void pushdown(int u){
      calc(tr[u<<1],tr[u].add,tr[u].mul);
      calc(tr[u<<1|1],tr[u].add,tr[u].mul);
      tr[u].add=0,tr[u].mul=1;
    }
    void build(int p,int l,int r){
      Node &u=tr[p];
      u.l=l,u.r=r;
      u.add=0,u.mul=1;
      if(l==r){
        u.sum=a[l];
        return;
      }
      int mid=(l+r)>>1;
      build(p<<1,l,mid);
      build(p<<1|1,mid+1,r);
      pushup(p); 
    }
    void modify(int p,int l,int r,int add,int mul){
      Node &u=tr[p];
      if(l<=u.l && u.r<=r){
        calc(u,add,mul);
        return;
      }
      pushdown(p);
      int mid=(u.l+u.r)>>1; // 这里不是修改的区间的中点，不是(l+r)>>1
      if(l<=mid) modify(p<<1,l,r,add,mul);
      if(r>mid) modify(p<<1|1,l,r,add,mul);
      pushup(p);
    } 
    ll query(int p,int l,int r){
      Node &u=tr[p];
      if(l<=u.l && u.r<=r)
        return u.sum;
      pushdown(p);
      int mid=(u.l+u.r)>>1;ll res=0; // 这里的mid同理
      if(l<=mid) res=query(p<<1,l,r);
      if(r>mid) res+=query(p<<1|1,l,r);
      res%=m;
      return res; 
    }
    int main() {
      ios::sync_with_stdio(0);
      cin.tie(NULL);
      cout.tie(NULL);
      cin>>n>>q>>m;
      for(int i=1;i<=n;i++)
        cin>>a[i];
      build(1,1,n);
      int op,x,y,k;
      while(q--){
        cin>>op>>x>>y;
        if(op==1){
          cin>>k;
          modify(1,x,y,0,k);
        }
        else if(op==2){
          cin>>k;
          modify(1,x,y,k,1);
        }
        else{
          cout<<query(1,x,y)<<endl;
        }
      }
      return 0;
    }
    ```

# 模板 & 总结

线段树一个结构体五个函数：

- Node
    
    - 包含区间端点，包含完备的信息，需要4倍空间
    

- Pushup
    
    - 利用左右儿子更新自己
    

- Pushdown
    
    - 将懒标记向下传
    

- build
    
    - 记录区间端点
    
    - 如果递归到叶节点，赋值返回
    
    - 否则左右子节点递归建树
    
    - Pushup
    

- update / modify / change
    
    - 如果区间完全被包含，更新信息，打上懒标记，返回
    
    - 否则 Pushdown
    
    - 左右子节点递归修改
    
    - Pushup
    

- query / ask （非 void）
    
    - 如果区间完全被包含，返回值或节点
    
    - 否则 Pushdown
    
    - 递归查询左右两侧
    
    - 合并左右两侧的查询结果并返回
