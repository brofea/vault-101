### 字符串转多个整数

作用：输入一行带空格的字符串，如”123 234 345\n”，将其转换为整数数组

- **法一：C++风格做法**
    
    ```C++
    \#include <iostream>
    \#include <sstream>
    \#include <string>
    \#include <vector>
    using namespace std;
    vector<int> arr;
    void input(){
        string str;
        getline(cin, str);
        istringstream iss(str);
        int num;
        while(iss>>num)
            arr.push_back(num);
    }
    int main(){
        input();
        for (int num:arr){
            cout << num << " ";
        }
    }
    ```
    
    基本思路：
    
    1. 使用`getline()` 读入整行带空格字符串进`str`
    
    1. 将`str`转换为`std::istringstream`输入流类型`iss`
    
    1. 将输入流输入进`num`，向`vector` 中压入数字
    
    1. 重复第三步
    
    - `main()`中的`for (int num:arr) cout << num << " ";`等价于`for(int i=0;i<arr.size();i++) cout<< arr[i] << " ";`
    

- **法二：C语言风格做法**
    
    ```C++
    char str[1000000];
    int arr[100000];
    int len = 0; // 数组长度
    void input(){
        // 使用 fgets 读取字符串
        fgets(str, sizeof(str), stdin);
        // 去掉字符串末尾的换行符
        str[strcspn(str, "\n")] = '\0';
        // 使用 strtok 分割字符串，并使用 sscanf 将每个部分转换为整数
        char *token=strtok(str, " ");
        while (token!=NULL) {
            sscanf(token,"%d",&arr[len]);
            len++;
            token=strtok(NULL," ");
        }
    }
    ```
    
    基本思路：
    
    1. 使用`fgets()`读入整行字符串
    
    1. 使用`strscpn()`找到行尾的换行符，替换成`’\0’`
    
    1. 用`strtok()`找到前方第一个空格的位置将其替换为`’\0’`，将前方的字符串的首地址赋给`token`，此时`token`指向的字符串相当于第一个整数
    
    1. 用`sscanf()`将`token`所在的字符串转换为整数，`len`向后移动
    
    1. 重复第4步，直到`token`变成空指针

# 二、时间计算

闰年规则：四年一闰，百年不闰，四百年又闰

### <ctime>的时间计算

```C
time_t is       // 从1900年至今走过的秒数
struct tm {
	int	tm_sec;		/* seconds after the minute [0-60] */
	int	tm_min;		/* minutes after the hour [0-59] */
	int	tm_hour;	/* hours since midnight [0-23] */
	int	tm_mday;	/* day of the month [1-31] */
	int	tm_mon;		/* months since January [0-11] */
	int	tm_year;	/* years since 1900 */
	int	tm_wday;	/* days since Sunday [0-6] */
	int	tm_yday;	/* days since January 1 [0-365] */
	int	tm_isdst;	/* Daylight Savings Time flag */
	long	tm_gmtoff;	/* offset from UTC in seconds */
	char	*tm_zone;	/* timezone abbreviation */
};
double difftime(time_t time1, time_t time2)
// 返回 time1 和 time2 之间相差的秒数 (time1-time2)
time_t mktime(struct tm *timeptr)
// 把 timeptr 所指向的结构转换为一个依据本地时区的 time_t 值
struct tm *localtime(const time_t *timer)
// timer 的值被分解为 tm 结构，并用本地时区表示
```

要注意日期是 1~31，而月份是 0~11，年份是 1900 + tm_year

在实际使用的时候，tm 可以这样定义：

```C++

```

## 初始化数组

`fill()` 函数，可以填充任意值

```C++
int arr[100];
fill(arr, arr + 100, 1);  // 把所有元素设为 1
vector<int> vec(100);
fill(vec.begin(), vec.end(), 1);  // 把 vector 设为 1
```

`memset()` 函数，填充内存

```C++
int arr[100];
memset(arr, 0, sizeof(arr));  // 把 arr 的所有字节设为 0
memset(arr, 0x3f, sizeof(arr));  // 把 arr 的所有字节设为 0x3f3f3f3f
// 下面这句只能给 int、short、ll 使用
memset(arr, -1, sizeof(arr));  // 把 arr 的所有字节设为 0xffffffff
```

要注意 `memset()` 函数只能够填充 0 或者对应长度的 16 进制数字，不能够填充任意值

在 C++ 中，使用 `fill()` 和使用 for 循环来初始化数组的时间消耗差不多，`memset()` 会比这二者快 20～40 倍，在开启 O2 优化的情况下，这三者速度几乎相同

## 单行字符串转整数

```C++
\#include <iostream>
\#include <sstream>
\#include <vector>
using namespace std;
int main() {
    string input;
    vector<int> a;

    // 使用std::getline读取整行输入
    cout << "Enter numbers, press Enter to finish:" << endl;
    getline(cin, input);

    // 使用std::istringstream来解析这一行输入
    istringstream stream(input);
    int temp;
    while (stream >> temp)
        a.push_back(temp);

    // 输出读取到的数字
    cout << "You entered:" << endl;
    for (int i=0;i<a.size();i++)
        cout << a[i] << " ";
    cout << endl;
    return 0;
}
```

核心思想：`<sstream>`头文件和 `std::istringstream`

```C++
\#include <iostream>
\#include <vector>
using namespace std;
int main() {
  int n,m,a=0;
  string s,res;
  cin>>n>>s>>m;
  for(int i=0;i<s.size();i++){
    if(isdigit(s[i])) a=a*n+s[i]-'0';
    else a=a*n+int(s[i])-'A'+10;
  }

    v.push_back(a%m);
    a/=m;
  }
```
