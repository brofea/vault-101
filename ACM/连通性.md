# 强连通分量

> [落谷 - 提单 强连通分量 & 缩点](https://www.luogu.com.cn/training/248579#information)  

## 引入

简单解释一些概念和定义：

联通：无向图中所有节点两两互相可达，则称这张图是连通的

强连通：有向图中所有节点两两**互相**可达，则称这张图是强连通的

- 强连通子图内任意两点 u, v 要满足 u→v, v→u 的路径都存在

连通分量：无向图中一个**最大**的联通的节点集合，即极大联通子图

强联通分量（SCC）：有向图中一个**最大**的强连通的节点集合，即极大强连通子图

- 一个图内可能有不止一个强联通分量，但是每个强联通分量都是极大的，无法增加点

拓扑图（DAG）：有向无环图

缩点：将图中所有强连通分量当作一个点重新建图

  

强连通分量最常见的应用是通过**缩点**构造拓扑图，这对于求最短/长路问题有帮助，只需要对每个强连通分量做 Floyd 再在拓扑图上用拓扑排序+DP就能快速求解最短路问题

## DFS 生成树

对一个有向图进行深度优先遍历后所有的边都会被归为 4 种类型的其中一种

**树边 (Tree edge)**

v 第一次被遍历，则 (u, v) 为树边

==**前向边 (Forward edge)**==

v 不是第一次被遍历且 v 在 u 的子树中，则 (u, v) 为前向边

==**后向边 / 反祖边 (Back edge)**==

v 不是第一次被比遍历且 v 是 u 的祖先，则 (u, v) 为后向边

==**横叉边 (Cross edge)**==

v 不是第一次被遍历且两个节点互不为祖先-后代关系

![[image 3.png|image 3.png]]

- 横叉边如果向右边（DFS 未搜索的地方）横插，其实是树边，所以不存在向右边横插的横插边

- 自环也属于后向边

- 时间戳判断法
    
    在 DFS 中标记树边后，可以使用时间戳来判断剩下的边属于哪三种类型。记录 `in[u]` 是第一次搜索到 u 的时间，`out[u]` 是 u 所有子树处理完成的时间
    
    - 前向边：`in[u] < in[v] < out[v] < out[u]`
    
    - 后向边：`in[v] < in[u] < out[u] < out[v]`
    
    - 横叉边：`in[v] < out[v] < in[u] < out[u]`
    

### DFS 树和 SCC 的关系

如何判断某一个点是否属于一个强连通分量？

情况 1：当前节点出发存在后向边，也就是可以走回祖先节点

情况 2：通过一条或多条横叉边走到某个节点后再通过后向边走到了祖先节点

还有一个性质：

> 如果结点 u 是某个强连通分量在搜索树中遇到的第一个结点，那么这个强连通分量的其余结点肯定是在搜索树中以  u 根的子树中。结点 u 被称为这个强连通分量的根。

## Tarjan’s SCC 算法

使用时间戳 `timestamp`，维护两个数组：

`dfn[u]` 表示第一次遍历到 u 的时间戳

`low[u]` 表示从 u 开始走，能走到的最小的时间戳的点的时间戳

- 若 u 是其所在强连通分量的根（DFS 树中的最高点），那么一定满足 `dfn[u] == low[n]`

维护一个栈，存放还没有确定归属的点，或者说潜在属于某个强连通分量的点。在遍历完一个节点时栈并不会弹出，只会在找完全一个强连通子图，也就是确定强连通分量的根后才会开始弹出

此外，还要用 `in_stk[N]` 数组记录该节点是否在栈内，方便判断后向边

  

**伪代码如下：**

访问当前节点 u，初始化其 `dfn` 和 `low`，将其入栈，标记已入栈

遍历 u 的所有邻边 (u, j)

如果 j 未访问，则递归访问，访问结束更新 `low`

如果 j 已访问，且 j 在栈中，更新 `low`

如果当前节点是一个强连通分量的根节点

循环出栈，标记出栈元素都为同一个强连通分量，直到 u 出栈

  

时间复杂度为 $O(n+m)$，其中 $n$ 为点数 $m$ 为边数，此外，Tarjan 算法天然支持自环和重边，无需特判

```C++
int h[N], e[M], ne[M], idx;
int dfn[N], low[N], timestamp;  // 时间戳
int stk[N], top;
bool in_stk[N];
int id[N], scc_cnt;  // 每个点所属分量编号

void tarjan(int u)
{
    dfn[u] = low[u] = ++ timestamp;
    stk[ ++ top] = u, in_stk[u] = true;
    
    for (int i = h[u]; ~i; i = ne[i])
    {
        int j = e[i];
        if (!dfn[j])
        {
            tarjan(j);
            low[u] = min(low[u], low[j]);
        }
        else if (in_stk[j])
            low[u] = min(low[u], dfn[j]);
    }
    
    if (dfn[u] == low[u])
    {
        ++ scc_cnt;
        int y;
        do {
            y = stk[top -- ];
            in_stk[y] = false;
            id[y] = scc_cnt;
        } while (y != u);
    }
}
```

- 链式前向星的头节点数组要初始化为 -1

- 这里有一份详细注释的模板代码
    
```C++
int h[N], e[M], ne[M], idx;
// 邻接表实现图的边：h[u] 是点 u 的链表头，e[i] 是第 i 条边的目标点，ne[i] 是下一条边的编号，idx 是边数计数器

int dfn[N], low[N], timestamp;
// dfn[u]: 点 u 被 DFS 首次访问的时间戳
// low[u]: 点 u 所能追溯到的最早祖先（包括自己）的时间戳
// timestamp: 全局时间戳，每访问一个点就加一

int stk[N], top;
bool in_stk[N];
// stk[]: 模拟栈，记录 DFS 过程中经过的点
// top: 当前栈顶指针
// in_stk[u]: 表示点 u 是否在栈中，用于判断后向边和 SCC 的边界

int id[N], scc_cnt;
// id[u]: 点 u 所属的强连通分量编号
// scc_cnt: 当前找到的强连通分量个数

void tarjan(int u)
{
    // 当前访问点 u，设置其 dfn 和 low，并将其入栈
    dfn[u] = low[u] = ++timestamp;
    stk[++top] = u;
    in_stk[u] = true;

    // 遍历 u 的所有邻边
    for (int i = h[u]; ~i; i = ne[i])
    {
        int j = e[i]; // j 是 u 的邻居节点

        if (!dfn[j])
        {
            // 如果 j 尚未访问，则递归访问 j
            tarjan(j);

            // 回溯时更新 u 的 low 值
            low[u] = min(low[u], low[j]);
        }
        else if (in_stk[j])
        {
            // 如果 j 已经访问过，且仍在栈中（即当前 SCC 路径上）
            // 则说明 u 可以通过一条后向边或横叉边指向 j，更新 low[u]
            low[u] = min(low[u], dfn[j]);
        }
        // 否则 j 已访问且不在栈中，说明是横叉边指向其他分量，不用管
    }

    // 如果当前节点 u 是某个 SCC 的根（即不能再向上追溯）
    if (dfn[u] == low[u])
    {
        ++scc_cnt;  // 找到一个新的强连通分量

        int y;
        do {
            y = stk[top--];      // 从栈中弹出
            in_stk[y] = false;   // 标记 y 不在栈中
            id[y] = scc_cnt;     // 把 y 归入当前的强连通分量
        } while (y != u);        // 一直弹到当前 SCC 的根节点 u
    }
}
```
    

## 类模板题

简单的计算并统计强连通分量：


> [落谷 P2863 [USACO06JAN] The Cow Prom S](https://www.luogu.com.cn/problem/P2863)  

AC 代码
    
统计的是点数大于 1 的强连通分量的个数，在找到强连通分量的根后要判断点数是否大于 1
    
```C++
\#include <bits/stdc++.h>
using namespace std;
const int N = 1e4 + 10;
const int M = 5e4 + 10;
int n, m;
int h[N], e[M], ne[M], idx;
int dfn[N], low[N], timestamp;
int stk[N], top;
bool in_stk[N];
int id[N], scc_cnt;
int ans = 0;
void add(int u, int v) {
  e[idx] = v;
  ne[idx] = h[u];
  h[u] = idx++;
}
void tarjan(int u) {
  dfn[u] = low[u] = ++timestamp;
  stk[++top] = u, in_stk[u] = true;
  for (int i = h[u]; ~i; i = ne[i]) {
    int j = e[i];
    if (!dfn[j]) {
      tarjan(j);
      low[u] = min(low[u], low[j]);
    } else if (in_stk[j])
      low[u] = min(low[u], dfn[j]);
  }
  if (dfn[u] == low[u]) {
    ++scc_cnt;
    int y, cnt = 0;
    do {
      ++cnt;
      y = stk[top--];
      in_stk[y] = false;
      id[y] = scc_cnt;
    } while (y != u);
    if (cnt > 1) {
      ans++;
    }
  }
}
int main() {
  ios::sync_with_stdio(0);
  cin.tie(0);
  cout.tie(0);
  cin >> n >> m;
  int u, v;
  memset(h, -1, sizeof h);
  for (int i = 1; i <= m; ++i) {
    cin >> u >> v;
    add(u, v);
  }
  for (int i = 1; i <= n; ++i) {
    if (!dfn[i]) {
      tarjan(i);
    }
  }
  cout << ans << endl;
  return 0;
}
```

## 后序操作

### **拓扑排序**

其实没有必要，连通分量编号的倒序就是拓扑序

### **缩点 & DAG 图 DP**

缩点建新图：确定新图中 scc_cnt 个点，遍历所有边 (u, v)，如果点 u, v 不在同一个连通分量中，则在新图中增加边 (`id[u]`, `id[v]`)

构建完 DAG 图后要求最短路或最长路，可以利用拓扑序递推，时间复杂度是线性的

下面这道模板题要求的是带点权的最长路问题，节点可以多次经过但是只算一次权重，非常合适用缩点 + DAG 图 DP 来做

> [落谷 - P3387 【模板】缩点  ](https://www.luogu.com.cn/problem/P3387)  

AC 代码
    
```C++
\#include <bits/stdc++.h>
using namespace std;
const int N = 1e4+10, M = 1e5+10;
int n, m, a[N]; // a[i]:点i的权值
int h[N], e[M], ne[M], idx = 1;
inline void add(int u, int v) {
  e[idx] = v;
  ne[idx] = h[u];
  h[u] = idx++;
}
// Tarjan
int dfn[N], low[N], timestamp = 0;
int stk[N], top = 0;
bool inStack[N];
int scc[N], scc_cnt = 0;
int val[N]; // DAG图的点权，等于内部点权之和
void tarjan(int u) {
  dfn[u] = low[u] = ++timestamp;
  stk[++top] = u, inStack[u] = true;
  for (int i = h[u]; ~i; i = ne[i]) {
    int v = e[i];
    if (!dfn[v]) {
      tarjan(v);
      low[u] = min(low[u], low[v]);
    } else if (inStack[v]) {
      low[u] = min(low[u], dfn[v]);
    }
  }
  if (dfn[u] == low[u]) {
    ++scc_cnt;
    int x;
    do {
      x = stk[top--];
      inStack[x] = false;
      scc[x] = scc_cnt;
      val[scc_cnt] += a[x];  // 注意累加内部点权，而非++
    } while (x != u);
  }
}
set<int> fa[N]; // 去重的反向DAG图
int f[N]; // f[i]:以i为终点的带点权最长路长度
int main() {
  ios::sync_with_stdio(0);
  cin.tie(0);
  cout.tie(0);
  memset(h, -1, sizeof h);
  cin >> n >> m;
  for (int i = 1; i <= n; ++i)
    cin >> a[i];
  int u, v;
  for (int i = 1; i <= m; ++i) {
    cin >> u >> v;
    add(u, v);
  }
  // 缩点建图
  for (int i = 1; i <= n; ++i)
    if (!dfn[i])
      tarjan(i);
  for (int i = 1; i <= n; ++i)
    for (int j = h[i]; ~j; j = ne[j])
      if (scc[i] != scc[e[j]])
        fa[scc[e[j]]].insert(scc[i]);
  // DAG DP
  int fmax = 0;
  for (int u = scc_cnt; u >= 1; --u) {
    f[u] = val[u];
    for (int v : fa[u])
      if (f[v] + val[u] > f[u])
        f[u] = f[v] + val[u];
    fmax = max(fmax, f[u]);
  }
  cout << fmax << '\n';
  return 0;
}
```
