# 二分和三分

## 整数二分

> [OI Wiki - 二分](https://oi-wiki.org/basic/binary/)  

二分查找：在单调序列中，不断将查找范围缩小一半，来快速定位目标值的算法

对于整数二分，当 `l` 和 `r` 相差 1 时，`mid` 可能等于 `l` 或 `r`，此时若执行 `l = mid` 或 `r = mid`，则会进入死循环

所以根据不同的情况，`mid` 归属不同，分为两种情况：

- 求第一个满足条件值：`mid` 属于左区间，计算时向左（下）取整
- 求最后的满足条件值：`mid` 属于右区间，计算时向右（上）取整

口诀是：属左取左，属右取右

### 求第一个满足条件值

由于要找第一个满足，当 `mid` 满足，答案肯定在左区间，此时舍弃右区间，但 `mid` 可能就是第一个，所以不能舍弃，要将 `mid` 归入左区间，即执行 `r = mid`，舍弃右半区间

自然地，`mid` 不满足时，答案肯定在右区间，此时舍弃左区间，执行 `l = mid + 1`

当 `l` 和 `r` 相差 1 时，由于 `mid` 属于左区间，计算时让 `mid` 向左边取整可以将 `l` 和 `r` 归入不同区间。否则 `l` 和 `r` 都属于左区间，右区间没有任何元素，抛弃右区间会进入死循环

```cpp
int l = , r = ;
while(l < r) {
  int mid = (l + r) >> 1;
  if(check(mid)) r = mid;
  else l = mid + 1;
}
// 结果取 l 或 r 都可
```

### 求最后的满足条件值

同理，要找最后一个满足，当 `mid` 满足，答案肯定在右区间，此时舍弃左区间，但 `mid` 可能就是最后一个，所以不能舍弃，要将 `mid` 归入右区间，即执行 `l = mid`，舍弃左半区间

自然地，`mid` 不满足时，答案肯定在左区间，此时舍弃右区间，执行 `r = mid - 1`

当 `l` 和 `r` 相差 1 时，由于 `mid` 属于右区间，计算时让 `mid` 向右边取整可以将 `l` 和 `r` 归入不同区间。否则 `l` 和 `r` 都属于右区间，左区间没有任何元素，抛弃左区间会进入死循环

```cpp
int l = , r = ;
while(l < r) {
  int mid = (l + r + 1) >> 1;
  if(check(mid)) l = mid;
  else r = mid - 1;
}
// 结果取 l 或 r 都可
```

在两个模板中，存在`bool check (int x);` 用于判断`mid`是否满足某种条

两个模板执行完后，`l` 和 `r` 相等，取任意一个即可

### 例题1

> [落谷 - P2249 【深基13.例1】查找](https://www.luogu.com.cn/problem/P2249)  

```C++
#include <iostream>
using namespace std;
int main() {
    const int N = 1e6 + 10;
    int a[N];
    int n, m, q;
    ios::sync_with_stdio(false);
    cin >> n >> m;
    for (int i = 0; i < n; i++) cin >> a[i];

    for (int u = 0; u < m; u++) {
        cin >> q;
        int l = 0, r = n - 1;
        while (l < r) {
            int mid = (l + r) >> 1;
            if (a[mid] >= q)
                r = mid;
            else
                l = mid + 1;
        }
        if (a[l] != q)
            cout << -1 << ' ';
        else
            cout << l + 1 << ' ';
    }
}
```

### STL 二分查找

> [CSDN - 【C++】详解 lower_bound 和 upper_bound 函数](https://blog.csdn.net/weixin_45031801/article/details/137544229)  

STL 中的 lower_bound 和 upper_bound 函数可以自定义比较器，传入仿函数或函数指针，使用这两个函数可以将例题简化为这样：

```C++
#include <algorithm>
#include <iostream>
#include <iterator>
#include <vector>
using namespace std;
int main() {
    vector<int> a;
    int n, m, q, input;
    ios::sync_with_stdio(false);
    cin >> n >> m;
    for (int i = 0; i < n; i++) a.push_back((cin >> input, input));
    for (int i = 0; i < m; i++) {
        auto iter = lower_bound(a.begin(), a.end(), (cin >> q, q));
        if (*iter == q)
            cout << distance(a.begin(), iter) + 1 << ' ';
        else
            cout << -1 << ' ';
    }
}
```

### 例题2

> [落谷 - P1102 A-B 数对](https://www.luogu.com.cn/problem/P1102)  

```C++
#include <algorithm>
#include <iostream>
#include <vector>
using namespace std;
int main() {
    int n, c, input;
    long long ans = 0;
    cin >> n >> c;
    vector<int> a;
    for (int i = 0; i < n; i++) a.push_back((cin >> input, input));
    sort(a.begin(), a.end());
    for (int num : a) {
        auto upit = upper_bound(a.begin(), a.end(), num + c);
        auto lowit = lower_bound(a.begin(), a.end(), num + c);
        ans += distance(a.begin(), upit) - distance(a.begin(), lowit);
    }
    cout << ans << endl;
}
```

- 注意使用 `long long` 存答案

## 浮点二分

和整数二分类似，但比较方式需要使用浮点数的比较，且需要设置精度 `eps`，当 `l` 和 `r` 的差值小于等于 `eps` 时，此时可以认为 `l` 和 `r` 是同一个数，迭代结束

使用这种比较方式也没有死循环的风险

### 基本步骤

```C++
const double eps = 1e-6;
double l = , r = ;
while (r - l > eps) {
    double mid = (l + r) / 2;
    if (check(mid))
        r = mid;
    else
        l = mid;
}
return l;
```

double 无法使用位运算除 2

存在`bool check (int x);` 用于判断`mid`是否满足某种条件

一般来说，要求精确到 n 位小数，可设置 `eps` 为 `1e(-n-2)`

### 例题3

一个简单的求平方根的程序

保证 $0\le n \le 100$ ，精确到两位小数

```C++
int main() {
    double n, l = 0, r = 100, eps = 1e-4;
    cin >> n;
    while (r - l > eps) {
        double mid = (l + r) / 2;
        if (mid * mid > n)
            r = mid;
        else
            l = mid;
    }
    cout << fixed << setprecision(2) << l << endl;
}
```

## 整数三分

用于在单峰函数或单谷函数中寻找最值，每次将区间分为三等份，舍弃最右边或最左边的三分之一，缩小搜索范围

可以证明的是，只要函数在目标区间内是严格的单峰/单谷函数，三分完全适用，哪怕左右极度不对称

### 单一最值

下面以 `f` 是单谷函数为例，当 `f(m1) > f(m2)` 时，舍弃左边的三分之一，执行 `l = m1 + 1`，否则执行 `r = m2 - 1`，舍弃右边的三分之一

至于当 `l` 和 `r` 太过接近时，我们不考虑得太复杂，直接将整个区间 `[l, r]` 遍历一遍，找到最小值即可

```cpp
int trisc(int l, int r) {
  while(l < r - 2) {
    int m1 = l + (r - l) / 3;
    int m2 = r - (r - l) / 3;
    if(f(m1) > f(m2)) l = m1 + 1;
    else r = m2 - 1;
  }
  int ans = l;
  for(int i = l + 1; i <= r; ++i) {
    if(f(i) < f(ans)) ans = i;
  }
  return ans;
}
```

### 求最值区间

在整数三分中，单峰/单谷函数可能形似平顶山/平底锅，存在多个最值点

先用三分找到一个最值点 `mm`，然后在 `[L, mm]` 区间中寻找第一个最值点，在 `[mm, R]` 区间中寻找最后一个最值点

```cpp
pair<int, int> trisc_range(int L, int R) {
  int mm = trisc(L, R), l, r;
  int fmm = f(mm);
  pair<int, int> res;

  l = L, r = mm;
  while(l < r) {
    int mid = (l + r) >> 1;
    if(f(mid) == f(mm)) r = mid;
    else l = mid + 1;
  }
  res.first = l;

  l = mm, r = R;
  while(l < r) {
    int mid = (l + r + 1) >> 1;
    if(f(mid) == f(mm)) l = mid;
    else r = mid - 1;
  }
  res.second = l;

  return res;
}
```
