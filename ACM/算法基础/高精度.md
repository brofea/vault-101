# 高精度计算

## 高精度加法

常考的高精度题：1、A + B 2、A - B 3、A * a 4、A / a  
其中大写的 A 和 B 位数上限大约为 $10^6$，小写的 a 一般不超过整型范围，以下内容讲的是第一种情况，高精度加高精度。

### 高精加基本思想

用数组倒序分别存放大整数的每一位，例如 `a[100]={9,8,7,6,5,4,3,2,1}`存放的就是 $123,456,789$这么一个数，倒序存放方便进位时添加数字

模拟列竖式计算是高精加的基本思想，每一位数字由另外两个数对应位数的相加，再加上上一位数的进位数

```cpp
vector<int> add(vector<int> &A, vector<int> &B)
{
    if (A.size() < B.size()) return add(B, A);

    vector<int> C;
    int t = 0;
    for (int i = 0; i < A.size(); i ++ )
    {
        t += A[i];
        if (i < B.size()) t += B[i];
        C.push_back(t % 10);
        t /= 10;
    }

    if (t) C.push_back(t);
    return C;
}
```

## 高精度减法

### 高精减基本思想

和高精度加法类似，进位所用的 `t` 变成了借位，最后将多余的 0 弹出，如果答案 vector 为空就再加上 0，如果 A 有可能小于 B，那就加上一个 `bool ifNeg` 函数判断正负

```cpp
vector<int> sub(vector<int> &A, vector<int> &B)
{
    vector<int> C;
    for (int i = 0, t = 0; i < A.size(); i ++ )
    {
        t = A[i] - t;
        if (i < B.size()) t -= B[i];
        C.push_back((t + 10) % 10);
        if (t < 0) t = -1;
        else t = 0;
    }

    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

### 例题2

> [落谷 - P2142 高精度减法](https://www.luogu.com.cn/problem/P2142)  

```cpp
#include <iostream>
#include <vector>
#include <cstdio>
using namespace std;
#define VI vector<int>
bool ifNeg(VI &a,VI &b){
    if(a.size()>b.size())
        return 0;
    else if(a.size()<b.size())
        return 1;
    for(int i=a.size();i>=0;i--){
        if(a[i]>b[i]) return 0;
        if(a[i]<b[i]) return 1;
    }
    return 0;
}
VI sub(VI &a,VI &b){
    if(ifNeg(a,b)==1)
        return sub(b,a);
    VI c;
    int t=0;
    for(int i=0;i<a.size();i++){
        t+=a[i];
        if(i<b.size()) t-=b[i];
        c.push_back((t+10)%10);
        if(t<0) t=-1;
        else t=0;
    }
    while(c.size()>0&&c.back()==0) c.pop_back();
    if(c.empty()) c.push_back(0);
    return c;
}
void print(VI &a){
    for(int i=a.size()-1;i>=0;i--)
        printf("%d",a[i]);
    printf("\n");
}
VI scan(VI &a){
    string input;
    cin>>input;
    for(int i=input.size()-1;i>=0;i--)
        a.push_back(input[i]-'0');
    return a;
}
int main() {
    VI a,b;
    scan(a);
    scan(b);
    VI c=sub(a,b);
    if(ifNeg(a,b)==1) printf("-");
    print(c);
}   
```

## 高精度乘法

### 高精度乘低精度

一般来说在高精度每一位数乘完被乘数时，long long 类的临时变量`t`仍然能存放，所以直接一次性让 A 的每一位数和 b 相乘得到`t`，`t%10`即为当前位数，`t/10` 即为下一位数的进位。要注意由于进位`t`有可能会非常大，所以要等`t`循环至 0 后再结束循环，最好要去除多余的 0

```cpp
vector<int> mul(vector<int> &A, int b)
{
    vector<int> C;

    int t = 0;
    for (int i = 0; i < A.size() || t; i ++ )
    {
        if (i < A.size()) t += A[i] * b;
        C.push_back(t % 10);
        t /= 10;
    }

    while (C.size() > 1 && C.back() == 0) C.pop_back();

    return C;
}
```

- 注意，高精度的所有算法中，刚进入循环时，`t` 都不会被直接赋值，而是使用`+=`或`-=` 运算，否则就销毁了上一位保留的进位

### 高精度乘高精度

> [落谷 - P1303 A*B Problem](https://www.luogu.com.cn/problem/P1303)  

```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;
const int MAXLEN=4000;
string multi(string a,string b){
    int res[MAXLEN];//用于储存未进位的积
    for(int i=0;i<MAXLEN;i++)
        res[i]=0;
    int lenA=a.size();
    int lenB=b.size();
    for(int i=0;i<lenA;i++)
        for(int j=0;j<lenB;j++)
            res[i+j]+=int(a[lenA-i-1]-'0')*int(b[lenB-j-1]-'0');
    int it=0;
    int minloop=lenA+lenB-2;
    while(res[it]>=10 || it<=minloop){
        res[it+1]+=res[it]/10;
        res[it]%=10;
        it++;
    }
    while(res[it]==0)
        it--;
    string ans="";
    for(int i=it;i>=0;i--)
        ans+=res[i]+'0';
    if(ans=="")
        return "0";
    return ans;
}
int main(){
    string a,b;
    cin >> a >> b;
    cout<< multi(a,b) <<endl;
    return 0;
}
```

基本思路：

列竖式计算，得到没有任何进位的结果，也就是说每一位数字都有可能是一个几十上百的数，所以用res数组储存，最后再进位，注意0做乘数的判断

例如：

$$\begin{array}{r}  
& & && 1 & 5 & 9 & 7 & 5 & 3 \\  
\times & && && & 1 & 2 & 3 & 4 \\  
\hline  
& && & 6 & 3 & 9 & 0 & 1 & 2 \\  
& && 4 & 7 & 9 & 2 & 5 & 9 \\  
& &3 & 1 & 9 & 5 & 0 & 6 \\  
+ & 1 & 5 & 9 & 7 & 5 & 3 \\  
\hline  
& 1 & 8 & 14 & 29 & 22 & 14 & 11 & 10 & 2 \\  
\end{array}$$

最后从个位开始进位并模10即可

## 高精度除法

### 高精度除低精度

高精度除法的算法和前面三种高精度略有不同，基本思路是模拟列竖式除法计算，先从高位开始除，循环中，先获取最高位，加在乘10倍的上一位的余数，除b得到当前位，在模b得到当前位的余数，循环结束，`r`即为$A\div b$，在函数内引用`r`以返回余数。

由于从高位开始计算，最后需要反转答案数组，然后去除高位0

```cpp
vector<int> div(vector<int> &A, int b, int &r)
{
    vector<int> C;
    r = 0;
    for (int i = A.size() - 1; i >= 0; i -- )
    {
        r = r * 10 + A[i];
        C.push_back(r / b);
        r %= b;
    }
    reverse(C.begin(), C.end());
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

### 例题4

> [落谷 - P1480 A/B Problem](https://www.luogu.com.cn/problem/P1480)  

```cpp
#include <algorithm>
#include <iostream>
#include <vector>
using namespace std;
#define VI vector<int>
VI div(VI& a, long long b, long long& r) {
    VI c;
    r = 0;
    for (int i = a.size() - 1; i >= 0; i--) {
        r = r * 10 + a[i];
        c.push_back(r / b);
        r = r % b;
    }
    reverse(c.begin(), c.end());
    while (c.size() > 1 && c.back() == 0) c.pop_back();
    return c;
}
void print(VI& a) {
    for (int i = a.size() - 1; i >= 0; i--) printf("%d", a[i]);
    printf("\n");
}
VI scan(VI& a) {
    string input;
    cin >> input;
    for (int i = input.size() - 1; i >= 0; i--) a.push_back(input[i] - '0');
    return a;
}
int main() {
    VI a;
    long long b, r;
    scan(a);
    cin >> b;
    VI c = div(a, b, r);
    print(c);
}
```
