# 树状数组 Fenwick Tree
  
> [OI Wiki - 树状数组](https://oi-wiki.org/ds/fenwick/)  

## 优势

树状数组着重解决两个操作

1. 单点修改，修改一个数 $O(\log n)$

1. 区间查询，比如求前缀和 $O(\log n)$

对比普通数组和前缀和数组，可以看到树状数组是这两者的折中思想

||单点修改|区间查询|
|---|---|---|
|普通数组|$O(1)$|$O(n)$|
|前缀和数组|$O(n)$|$O(1)$|
|树状数组|$O(\log n)$|$O(\log n)$|

## 设计思路

树状数组利用二进制思想，假设 $x$ 的二进制表示中有 $k$ 个 $1$，他们的位置为 $i_1,i_2,i_3,\dots,i_k$，（$i_1$ 到 $i_k$ 递增且 $k\le\log_2 x$）即：

$$x=2^{i_k}+2^{i_k-1}+\dots+2^{i_2}+2^{i_1}$$

此时如果要求数组中区间 $(0,x]$ 的值，可以划分成以下区间：

$(x-2^{i_1},x]$，包含 $2^{i_1}$ 个数$(x-2^{i_1}-2^{i_2},x-2^{i_1}]$，包含 $2^{i_2}$ 个数$(x-2^{i_1}-2^{i_2}-2^{i_3},x-2^{i_1}-2^{i_2}]$，包含 $2^{i_3}$ 个数$\dots$$(0,x-2^{i_1}-2^{i_1}-\dots-2^{i_k-1}]$，包含 $2^{i_k}$ 个数

这样一来，求任意区间时只取要求 $\log x$ 次区间和就可以求出所需区间

不难发现，每一个区间的右端点都是 $x$ 进行 lowbit 运算，此时讨论区间为整数，就可以将区间表示为：

$[x-lowbit(x)+1,x]$$[y-lowbit(y)+1,y],y=x-lowbit(x)$$\dots$

所以就可以用一个数组表示一个区间：

$$c[x]=a[x-lowbit(x),x]$$

## 原理

### 如何通过父节点找子节点

> 先将其减一，得到一段连续的 1 区间，再一次次去掉最后一位 1 直到 1 区间消失，即循环进行 $x-lowbit(x)$

例1：c[16]，二进制数为 10000，减去 1 变为 15，即 1111

- 第一次减去末位 1，得到区间 $((1110)_2,(1111)_2]=(14,15]$，即 $c[15]$

- 第二次减去末位 1，得到区间 $((1100)_2,(1110)_2]=(12,14]$，即 $c[14]$

- 第三次减去末位 1，得到区间 $((1000)_2,(1100)_2]=(8,12]$，即 $c[12]$

- 第四次减去末位 1，得到区间 $((0000)_2,(1000)_2]=(0,8]$，即 $c[8]$

即 $c[16]=a[16]+c[15]+c[14]+c[12]+c[8]$，和图中是一致的，$c[16]$ 连接了五根红线

例2：c[12]，二进制数为 1100，减 1 得 1011，只需要操作最后两位 1

区间为$((1010)_2,(1011)_2],((1000)_2,(1010)_2]=(10,11],(8,10]$

即：$c[12]=c[10]+c[11]+a[12]$

### 如何通过子节点找到父节点

不难发现，通过父节点找子节点时，每一次区间的右节点总是有一个固定位置的 1，这个 1 的位置就是一开始的连续 1 的最高位，所以通过子节点找父节点就是逆运算

> 从低位向高位找第一段连续的 1 的区间，然后将高于连续 1 区间的一位 0 变为 1，然后将后面的数都改为 0，即循环进行 $x+lowbit(x)$

假设子节点为 $x=(00101110)_2$，找到第一段连续 1 区间，即 111，然后将前一位的 0 变为 1，得到 $(00110000)_2$，这个操作等同于 $x+lowbit(x)$

通过子节点找父节点很重要，这是进行树状数组修改元素的基础，每次修改数组 a 都要往上找父节点进行更新

## 实现

### 更新

为了将位置 $x$ 加上 $c$ 根据子节点找父节点的思想，循环进行 $+lowbit(x)$ 的操作，将需要操作的位置及其每个父节点都加上 $c$，实现如以下模板的 update() 函数

### 查询

为了查询前 $x$ 个数的和，根据父节点找子节点的思想，循环进行 $-lowbit(x)$ 的操作，即每次把最后一位去掉，就能找到所有的子节点，对子节点求和就是答案，如以下模板的 query() 函数

- `tr[i]` 表示以`i` 结尾的长度为 `lowbit(i)` 的区间，这个区间的和

### 初始化

如果用 update() 函数直接加入空数组，那么时间复杂度是 $O(n\log n)$，可以用一种 $O(n)$ 的初始化方法：

树状数组的定义简单来说就是 $c[x]=\sum_{i=x-\text{lowbit}(x)+1}^xa[i]$，可以先对 $a[i]$ 求前缀和 $s[i]$ 那么 $c[x]=s[x]-s[x-\text{lowbit}(x)]$

### 区间查单点改

```C++
int lowbit(int x)
{
    return x & -x;
}

void update(int x, int c)  // 位置x加c
{
    for (int i = x; i <= n; i += lowbit(i)) tr[i] += c;
}

int query(int x)  // 返回前x个数的和
{
    int res = 0;
    for (int i = x; i; i -= lowbit(i)) res += tr[i];
    return res;
}
```

- 要注意每次循环是加减 lowbit(i) 而不是 lowbit(x)

模板题（区间查单点加）

- [https://www.luogu.com.cn/problem/P3374](https://www.luogu.com.cn/problem/P3374)

```C++
#include <bits/stdc++.h>
using namespace std;
const int N = 5e5+10;
int n,tr[N],a[N],pre[N];
int lowbit(int x){
    return x&-x;
}
void update(int x,int c){
    for(int i=x;i<=n;i+=lowbit(i))
    tr[i]+=c;
}
int query(int x){
    int res=0;
    for(int i=x;i;i-=lowbit(i))
    res+=tr[i];
    return res;
}
int main() {
    int m,op,x,y,k,tmp;
    cin>>n>>m;
    for(int i=1;i<=n;i++){
    cin>>a[i];
    pre[i]=pre[i-1]+a[i];
    }
    for(int i=1;i<=n;i++)
    tr[i]=pre[i]-pre[i-lowbit(i)];
    for(int i=1;i<=m;i++){
    cin>>op;
    if(op==1){
        cin>>x>>k;
        update(x,k);
    }
    if(op==2){
        cin>>x>>y;
        cout<<query(y)-query(x-1)<<endl;
    }
    }
}
```

## 扩展

### 单点查区间改（差分）

前面的树状数组可以操作单点，查询区间，如果要操作区间，查询单点，可以使用将原数组先处理成一个差分数组，然后将这个差分数组构建成树状数组

如果有了原数组 $a[~]$ 的差分数组 $b[~]$，要使 a[L~R] 都加上 $c$，相当于 b[L]+=c, b[R+1]-=c

如果要求 a[x] 相当于求 b[1~x] 的和，和基础的树状数组的查询完全一致

模板题如下

- [Luogu P3368](https://www.luogu.com.cn/problem/P3368)

```C++
#include <bits/stdc++.h>
using namespace std;
const int N = 5e5+10;
int tr[N],n;  // 树状数组和个数
int a[N],b[N];// 原数组和差分数组
int lowbit(int x){
    return x&-x;
}
void update(int x,int c){
    for(int i=x;i<=n;i+=lowbit(i))
    tr[i]+=c;
}
int query(int x){
    int res=0;
    for(int i=x;i;i-=lowbit(i))
    res+=tr[i];
    return res;
}
signed main() {
    ios::sync_with_stdio(0);
    cin.tie(NULL);
    cout.tie(NULL);
    int m,op,x,y,k;
    cin>>n>>m;
    for(int i=1;i<=n;i++){
    cin>>a[i];
    b[i]=a[i]-a[i-1];
    }
    // 用差分数组的前缀和（即原数组）初始化树状数组
    for(int i=1;i<=n;i++)
    tr[i]=a[i]-a[i-lowbit(i)];
    // m次查询
    for(int i=1;i<=m;i++){
    cin>>op;
    // 将区间[x,y]加上k
    if(op==1){
        cin>>x>>y>>k;
        update(x,k);
        if(y+1<=n) update(y+1,-k);
    }
    // 查询x位置的值
    if(op==2){
        cin>>x;
        cout<<query(x)<<endl;
    }
    }
    return 0;
}
```

### 区间查区间改

如果又要区间查询，又允许区间加，使用差分数组可以解决区间加的问题，现在考虑区间和

求 $a[1～x]$ 的和就是求如下式子

$$\sum_{i=1}^xa[i]=\sum_{i=1}^x\sum_{j=1}^ib[j]$$

要求的就是右图中蓝色的和，可以用 $x+1$ 个最后一行减去红色部分，红色部分可以竖着求和，变形为：

$$(x+1)a[x]-(xb[x]+\dots+2b[2]+1b[1])$$

不难看出后面一部分就是 $ib[i]$ 的前缀和，前面 $a[x]$ 就是 $b[~]$ 的前缀和，也就是说，可以维护两个树状数组来计算区间和

$$\begin{array}{}  
\textcolor{red}{b_1}&\textcolor{red}{b_2}&\textcolor{red}{b_3}&\textcolor{red}{\dots}&\textcolor{red}{b_x}\\  
\textcolor{blue}{b_1}&\textcolor{red}{b_2}&\textcolor{red}{b_3}&\textcolor{red}{\dots}&\textcolor{red}{b_x}\\  
\textcolor{blue}{b_1}&\textcolor{blue}{b_2}&\textcolor{red}{b_3}&\textcolor{red}{\dots}&\textcolor{red}{b_x}\\  
\textcolor{blue}{\vdots} &\textcolor{blue}{\vdots} &\textcolor{blue}{\vdots} &\textcolor{blue}{\ddots}&\textcolor{red}{\vdots} \\  
\textcolor{blue}{b_1}&\textcolor{blue}{b_2}&\textcolor{blue}{b_3}&\textcolor{blue}{\dots}&\textcolor{blue}{b_x}\\  
\end{array}$$

更新 b[i] 树的时候，$tr[l]$ 加 $c$，$tr[r+1]$ 减 $c$

更新 ib[i] 树的时候，$tr2[l]$ 加 $c\times l$，$tr2[r+1]$ 减 $c(r+1)$

**模板**

```C++
int t1[MAXN], t2[MAXN], n;

int lowbit(int x) { return x & (-x); }

void add(int k, int v) {
  int v1 = k * v;
  while (k <= n) {
    t1[k] += v, t2[k] += v1;
    // 注意不能写成 t2[k] += k * v，因为 k 的值已经不是原数组的下标了
    k += lowbit(k);
  }
}

int getsum(int *t, int k) {
  int ret = 0;
  while (k) {
    ret += t[k];
    k -= lowbit(k);
  }
  return ret;
}

void add1(int l, int r, int v) {
  add(l, v), add(r + 1, -v);  // 将区间加差分为两个前缀加
}

long long getsum1(int l, int r) {
  return (r + 1ll) * getsum(t1, r) - 1ll * l * getsum(t1, l - 1) -
         (getsum(t2, r) - getsum(t2, l - 1));
}
```
