# 面向对象编程概述

[面向对象程序设计](https://zh.wikipedia.org/wiki/%E9%9D%A2%E5%90%91%E5%AF%B9%E8%B1%A1%E7%A8%8B%E5%BA%8F%E8%AE%BE%E8%AE%A1)，港台称物件导向程式设计，英文称 Object-Oriented Programming，简称 OOP

核心思想：将现实世界中的事物抽象成对象，程序通过对象之间的交互来实现功能

为了达成这个目标，面向对象程序设计引入了以下几个重要的特征

- 分享非面向对象程序前身语言：如 C 语言的函数、变量、流程控制等
- 抽象（Abstraction）：将复杂的现实世界事物抽象成对象
- 类与对象（Class and Object）
  - 类（Class）：定义了一件事物的抽象特点。类的定义包含了数据的形式以及对数据的操作
  - 对象（Object）：是类的实例（Instance）
- 动态配置与消息传递机制
- **封装（Encapsulation）**：将数据和对数据的操作封装在一起，隐藏内部实现细节
- **继承（Inheritance）**：子类可以继承父类的属性和方法，促进代码重用
- **多态（Polymorphism）**：同一操作作用于不同的对象，可以有不同的解释和不同的执行结果

其中最后三个特征是 OOP 的核心特征，统称 OOP 三大特性
