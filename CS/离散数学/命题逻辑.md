# 命题逻辑
## 基础
### 命题
概念和定义
- 真值 / 值：命题到真值集合的映射
- 命题：唯一真值陈述句
- 原子命题 / 简单命题：唯一真值简单陈述句
- 命题常项 / 命题常元：真值确定的简单命题
- 命题变项 / 命题变元：真值可变化的简单陈述句
- 复合命题：简单命题+联结词，本质是真值函数
- 否定式：复合命题“非p”，否定联结词为 $\lnot$
- 合取式：复合命题“p并且q”，合取联结词为 $\land$
- 析取式：复合命题“p或q”，析取联结词为 $\lor$
- 蕴含式：复合命题“p则q”，p为前件q为后件，蕴含联结词为 $\to$
- 等价式：复合命题“p当且仅当q”，等价联结词为 $\leftrightarrow$
注意事项
- 命题变项不是命题，它只是一个空位
- “p则q”为假当且仅当p为真q为假
- 运算优先级顺序：$\lnot \land \lor \to \leftrightarrow$
