# 计算机网络：传输层

## IP

### 常考 IP 端口
#### 应用层服务
| 服务/协议 | 默认端口 | 传输层协议 | 备注 |
|---|---|---|---|
| HTTP | 80 | TCP | |
| HTTPS | 443 | TCP |  |
| DNS | 53 | UDP / TCP | 域名解析一般查询用 UDP，区域传输（主备服务器同步）用 TCP |
| DHCP | 67 / 68 | UDP | 动态主机配置，服务器用 67，客户端用 68 |
| FTP | 20 / 21 | TCP | 21 传命令，20 传文件 |
| SSH | 22 | TCP | 安全外壳，远程登录 |
| Telnet | 23 | TCP | 明文远程登录（不安全） |
#### 邮件、网络管理与文件共享
| 服务/协议 | 默认端口 | 传输层协议 | 作用与考点 |
|---|---|---|---|
| SMTP | 25 | TCP | 简单邮件传输协议，用于发送邮件 |
| POP3 | 110 | TCP | 邮局协议第三版，用于接收（下载）邮件 |
| IMAP | 143 | TCP | 交互式邮件存取协议，另一种接收邮件协议（支持同步状态） |
| TFTP | 69 | UDP | 简单文件传输协议因为简单、开销小，所以用 UDP |
| SNMP | 161 / 162 | UDP | 简单网络管理协议管理站用 162 接收 Trap 报文，代理用 161 |
