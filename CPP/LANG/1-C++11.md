# C++11 新增特性

## Lambda 表达式

一种匿名函数对象，相当于用下面这个结构来代替了函数名

```cpp
[捕获列表](参数列表) -> 返回类型 { 函数体 }
```

### 基本用法

比如说对结构体 `ST` 中的元素进行排序

```cpp
std::sort(v.begin(), v.end(), [](ST& a, ST& b) {
    return a.first < b.first;
});
```

其中，`[](ST& a, ST& b) { return a.first < b。first;}` 代替了原本的函数名 `cmp`

再比如，在二分排序中

```cpp
while(l < r) {
    int mid = (l + r + 1) >> 1;
    if([&](int k) {
        each if (k * d[i] > a[i].x) return false;
        return true;
    }(mid)) l = mid;
    else r = mid - 1;
}
```

其中，`[...](...){...}` 这一串结构就代替了原本的 `check` 函数，最后要调用该函数，要像 `check(mid)` 一样在后面加上 `(mid)` 来调用

### 捕获列表

也称 Capture 子句，决定了 Lambda 对外部变量的访问方式

```cpp
[]         // 不捕获
[=]        // 按值捕获
[&]        // 引用捕获
[=, &a]    // 默认按值捕获，a 引用捕获
[&, a]     // 默认引用捕获，a 按值捕获
[a, &b, c] // ac 按值捕获，b 引用捕获
```

### 返回类型

` -> 返回类型` 是可选的，编译器会自动推倒返回类型，但有歧义时必须写，比如同时返回了了 `int` 和 `double`，编译器无法推导

```cpp
[](int a, int b) -> double {
    if(a > b) return a;
    else return 1.0 * a / b;
}
```

### 赋名

Lambda 表达式可以赋给一个变量，变量的类型是 `std::function`，但多数情况用 `auto` 自动推导

```cpp
std::function<bool(ST&, ST&)> cmp1 = [](ST& a, ST& b) {
    return a.first < b.first;
};
std::sort(v.begin(), v.end(), cmp1);

auto cmp2 = [](ST& a, ST& b) {
    return a.first < b.first;
};
std::sort(v.begin(), v.end(), cmp2);
```
