# Gin 路由

## HTTP 方法

Gin 提供了直接映射到 HTTP 动词的方法

|方法|典型 REST 用途|
|---|---|
|GET|获取资源|
|POST|创建新资源|
|PUT|替换现有资源|
|PATCH|部分更新现有资源|
|DELETE|删除资源|
|HEAD|与 GET 相同但不返回响应体|
|OPTIONS|描述通信选项|

标准格式是

```go
var router = gin.Default()
router.METHOD("路径", 处理函数)
```

以 GET 方法为例：

```go
var r = gin.Default()
r.GET("/ping", func(c *gin.Context) {
    c.JSON(200, gin.H{"method": "GET"})
})
r.Run()
```

也可以将函数定义为一个变量：

```go
func pingHandler(c *gin.Context) {
    c.JSON(200, gin.H{"method": "GET"})
}
var r = gin.Default()
r.GET("/ping", pingHandler)
r.Run()
```

其中 router 的类型为 `*gin.Engine`，处理函数的类型为 `func(*gin.Context)`，`gin.Context` 包含了请求和响应的所有信息

## gin.Context

`gin.Context` 是 Gin 框架中处理 HTTP 请求和响应的核心结构体，它封装了 HTTP 请求的所有信息，提供了许多方法来方便地处理请求和生成响应
