## 创建一个简单的 Gin 应用

创建 `main.go` 文件，并添加以下代码：

```go
package main

import "github.com/gin-gonic/gin"

func main() {
  // 创建一个默认的 Gin 路由器
  router := gin.Default()
  // 定义一个 GET 请求的路由，路径为 /ping
  // 当访问 /ping 时，返回一个 JSON 响应，包含一个 message 字段，值为 "pong"
  router.GET("/ping", func(c *gin.Context) {
    c.JSON(200, gin.H{
      "message": "pong",
    })
  })
  // 启动 Gin 服务器，监听默认的 8080 端口
  router.Run()
}
```

使用 `go run main.go` 运行该应用，此时访问 [http://localhost:8080/ping](http://localhost:8080/ping) 将会得到如下响应：

```json
{
  "message": "pong"
}
```

## [官方 examples](https://github.com/gin-gonic/examples#)

这些示例涵盖了 Gin 的各种功能和用法，包括：

1. 入门核心 \
basic、group-routes、versioning \
作用：路由定义、路由分组、API 版本化等最基础能力。

1. 请求处理与数据绑定 \
form-binding、file-binding、custom-validation、struct-lvl-validations \
作用：把请求参数绑定到结构体、做字段级/结构级校验。

1. 页面与资源 \
template、assets-in-binary、http-pusher \
作用：模板渲染、静态资源处理、资源推送相关实践。

1. 常见 Web 功能 \
cookie、upload-file、cors-middleware、ratelimiter \
作用：Cookie、单/多文件上传、跨域、限流。

1. 服务稳定性与生命周期 \
graceful-shutdown \
作用：优雅停机、请求收尾、服务退出流程。

1. 实时通信 \
server-sent-event、websocket、realtime-chat、realtime-advanced \
作用：SSE 与 WebSocket，从简单聊天到多房间实时系统。

1. 网络与协议进阶 \
reverse-proxy、forward-proxy、http2、grpc \
作用：代理模式、HTTP/2、Gin 与 gRPC 组合。

1. 安全与生产化能力 \
secure-web-app、auto-tls、oidc、otel、new_relic \
作用：HTTPS/TLS、OIDC 登录、可观测性与 APM。

1. 云平台相关 \
app-engine \
作用：Gin 在 GAE 等平台的部署示例。
