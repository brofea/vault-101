# Vault 101

这是 brofea 的个人笔记仓库，存放 CS 专业相关笔记，名字源于游戏《辐射3》

## 仓库结构

- ACM - 存放竞赛及数据结构与算法相关笔记，分类参考 [OI Wiki](https://oi-wiki.org/)
  - docs - 存放 XCPC 赛时用的资料，比如板子和常见英文单词
- CS - 存放计算机科学相关专业笔记
- GO - 存放 Go 语言及相关技术栈笔记
- JAVA - 存放 Java 语言及相关技术栈笔记

## 文档规范

- Markdown 遵循 [markdownlint](https://github.com/DavidAnson/markdownlint) 默认规则，不使用非标准语法糖如 Obsidian 的 `[[]]`
- 非竞赛的 C++ 代码遵循 [Google C++ Style Guide](https://google.github.io/styleguide/cppguide.html)，竞赛的 C++ 代码也大致遵循该规范，但可能出现压行等不规范写法

## 快速上手

使用 `git clone` 克隆此文件夹，用 [VSCode](https://code.visualstudio.com/) 或 [Typora](https://typora.io/) 等 Markdown 编辑器打开文件夹即可

在 VSCode 中 commit 可以输入 `now` 并按下 Tab 键，输入和 Obsidian git 插件一样的 commit message
