# Vault 101

这是 brofea 的 Obsidian 仓库，存放 CS 专业相关笔记，名字源于游戏《辐射3》

## 仓库结构

- ACM - 存放竞赛及数据结构与算法相关笔记，也有比赛用的 Typst 板子存放在 /docs 下
- CS - 存放计算机科学相关专业笔记

## 文档规范

- Markdown 遵循 [markdownlint](https://github.com/DavidAnson/markdownlint) 默认规则
- C++ 遵循 [Google C++ Style Guide](https://google.github.io/styleguide/cppguide.html)

## 快速上手

使用 `git clone` 克隆此文件夹，用 [Obsidian](https://obsidian.md/) 打开文件夹即可

在 VSCode 中 commit 可以输入 `now` 并按下 Tab 键输入和 Obsidian git 插件一样的 commit message
